Traffic Jam In progress


Traffic problems are my personal headache so I decided to study them. Firsts things first, I need a traffic simulator to test and see where is the problem and how to fix it, but to build an efficient simulator is a big deal. Here we have the project roadmap:

  • Building the simulator
    1. Defining the different actors: people, buildings, streets
    2. Give the actors a behaviour
    3. Pathfinding

  • Understanding crowd behaviour
    1. Running tests on different situations
    2. Use machine learning to set the best conditions
    3. Analize the results
