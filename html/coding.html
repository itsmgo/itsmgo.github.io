The Game of Life Completed


The famous John Conway's game of life is a cellular simulation that occurs in a 2D grid. Each cell of the grid can be either dead or alive. Given certain living conditions, the grid evolves through time. Those conditions depend on the cell's neighbours and they are the following:

  • A cell dies when:
    1. There are 4 or more neighbours alive. The cell dies of overpopulation
    2. There are 1 or less neightbours alive. The cell dies of isolation
  • A cell comes to life when it has 3 neightbours alive
  • A cell remains dead/alive when it has 2 neighbours alive
I coded The Game of Life in Python and made an application using the GTK3+ library. I also made a visual representation using matplotlib library. Here you can browse the code:


And here you can download the Python app. You can pass 4 arguments to the program, X size, Y size, fife factor (between 0 and 1), and simulation velocity (miliseconds to refresh each frame). Example to run the program from terminal:

cd /path/to/file/directory
python gameoflife.py 25 25 0.5 150
